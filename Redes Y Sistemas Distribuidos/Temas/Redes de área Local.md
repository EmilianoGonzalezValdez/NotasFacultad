
# Redes de área Local



Una LAN es una red operada privadamente dentro de un edificio o una casa, suelen usarse en organizaciones y si es usada por una compañía se les llama redes empresariales.



Suelen interconectar computadoras, notebooks, celulares, impresoras, electrónicos del hogar, etc.

Estas LAN pueden ser:

- Inalambricas, se conectan por una estacion base (acces point) siguiendo el protocolo 802.11 (wifi)

- Ethernet, Se conectan por cables a un switch (conmutador)

> Si un dispositivo manda un mensaje, todos los demás lo reciben(No necesariamente fue enviado para todos)

Puede pasar que los mensajes enviados no lleguen o o se dañen, esto pasa si sucede una colisión, es decir, cuando mas de una maquina manda un mensaje al mismo tiempo, ocasionando que estos mensajes colisionen entre si y se dañen. Estas colisiones pueden ser evitadas, detectadas y tratadas.
