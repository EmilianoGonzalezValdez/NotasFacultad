
# Capa de aplicación



En la capa de aplicación tenemos las **aplicaciones de red**, cada una de estas ofrece un servicio especifico con su propia interfaz con el usuario

Para desarrollar aplicaciones de red tenemos 2 opciones:

1. Especificar la comunicacion usando una API

2. Apoyarnos en middlewares

    Un middleware provee servicios al software de aplicacion



# *Capa de aplicación: TCP/IP*

La capa de aplicación en TCP/IP contiene varios protocolos de nivel mas alto: transferencia de archivos (FTP), correo electrónico (SMTP), para resolución de nombres de host en sus direcciones de red (DNS), para la web (HTTP), etc

En la capa de aplicacion es muy importante la web y los protocolos que la soportan, tales como HTTP,HTTPS,DNS,etc
