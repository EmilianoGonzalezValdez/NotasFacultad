
# Aplicaciones Web



Una aplicación web es un sotware, el cual, retorna y almacena informacion usando scripts del lado del servidor, mientras que del lado del browser ejecutan scripts para diversas tareas. SU objetivo principal es que el usuario realice una o más tareas



 En una **Aplicación web Social** Los usuarios pueden crear y compartir contenido, ademas de comunicarse entre si mediante posts, comentarios y mensajes
