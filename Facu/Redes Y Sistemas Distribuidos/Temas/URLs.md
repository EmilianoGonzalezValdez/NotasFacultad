
# URLs



Para lograr nombrar una pagina web se utilizan las **URLs**(**Localizadores Uniformes de Recursos**)**.**

**Partes de un URL:**

- Nombre del protocolo

- Nombre de domiino de host que contiene la pagina

- Nombre del archivo que contiene la pagina(path al archivo)



El problema es que una URL como esta no basta para especificar la página dinamica deseada o el programa que obtiene los adtos deseados. Esto debido a que es necesario tener parametros en nuestro pedido HTTP

Una solución seria que el URL contenga el nombre del programa y párametros, los cuales son ingresados por medio de formulario HTML. Otra solución seria que los parametros se ingresan separados por & en un campo especial del pedido HTTP llamado **Cuepo de la entidad** 

En particular tenemos que las URL tienen un tamaño maximo, por lo cual la solución 1 no es tan rentable, siendo la solucion 2 la mas util en esos casos
