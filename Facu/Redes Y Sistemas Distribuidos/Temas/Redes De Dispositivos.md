
# Redes De Dispositivos

 Una red de dispositivos es una estructura compuesta por varios dispositivos, que llamaremos nodos interconetados 

#### ¿Con que motivo?

- Bueno pues vamos a decir que estos nodos vana  estar interconectados si pueden intercambiar información entre ellos. De esta manera logran compartir recursos como almacenamiento, aplicaciones y servicios facilitando el uso de cualquier nodo conectado

Esta conexión puede hacerse físicamente o  de forma inalambrica, PERO cabe recalcar que las comunicaciones entre los nodos siguen protocolos específicos para asegurar la correcta y eficiente transmisión de datos 

## ¿Que redes de dispositivos vamos a ver en la materia?

[[La Internet]]: Sirve para conectar computadoras(y algún que otro dispositivo) entre si por medio de proveedores de Internet

[[La nube]]: Provee servicios a personas o organizaciones como almacenamiento y ejecución de aplicaciones mediante una red de servidores interconectados 

[[La Internet de las cosas(IOT)]]: Son dispositivos específicos llamados dispositivos IOT (internet of things) los cuales se conectan entre si y con sistemas en la nube mediante el uso de Internet

[[Red Blockchain]]: Es un conjunto de nodos interconectados usados para mantener y validar un registro digital descentralizado de transacciones(Son datos repartidos por todo el mundo que tienen info de transacciones)



### Para lograr entender estas redes de dispositivos es necesario:

- Entender su propósito

- Entender como se conectan los nodos entre si y su organización

- Entender el sistema operativo de red, el cual se usa para gestionar el uso de recursos de la red y mantener la comunicación entre los nodos

- Entender los protocolos mas importantes que componen al sistema operativo de la red
