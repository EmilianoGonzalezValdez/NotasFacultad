
# Capa de Red



 La capa de red se encarga de:

- Almacenamiento y reenvio de mensajes. 

- Control de congestion

- Resolver problemas de envio de mensaje entre distintas tecnologias

- Enrutamiento(algoritmos de enrutamiento)



# La Capa de Red:TCP/IP

**Capa de interred:**

- Permite que los host inyecten paquetes dentro de cualquiera de las redes de interred

- Los paquetes diferentes entre 2 host viajan a su destino de manera independiente

    - Los paquetes pueden llegar en u orden distinto al enviado

    - Las capas mas altas deberian ordenarlos

Los paquetes que se envian son Paquetes IP con su propio formato.

Para hacer enrutamiento se utilizan protocolos como OSPF y BGP
