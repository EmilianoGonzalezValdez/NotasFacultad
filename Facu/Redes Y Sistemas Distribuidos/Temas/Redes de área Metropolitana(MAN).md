
# Redes de área Metropolitana(MAN)



Una MAN cubre una ciudad. Pueden ser:

- De cable, los cuales se basan en la red de TV por cable

    Unen las casas por un cable coaxial

    Se utilizan elementos de conmutacion para comunicar viviendas de diferentes cables coaxiales

    Estos elementos de conmutacion se unen por fibra optica

    Se usa un cable modem para que el servicio no sea antisimetrico

    Se tiene un divisor entre el cable de TV y el cable modem

    En una red se conectan las casas  a un proveedor de servicios de Internet, estas casas comparten el acceso a  un cable headend

- De Wimax, los cuales son redes inalambricas de alta velocidad 

    Hay una estación base que permite enviar paquetes por el aire 

    La estación base se conecta a internet

    Se puede acceder a la red WIMAX desde computadoras en casas hasta autos en movimiento
